\section*{Reproducibility}
The pipeline is driven by a single YAML configuration whose hash is recorded in
every result artifact, and every number in this report is read programmatically
from those artifacts rather than transcribed by hand. Random seeds are fixed
and propagated to NumPy, scikit-learn and PyTorch.
